개발 사항
기획의 원본 데이터인 엑셀파일을 서버가 사용할 수 있게 json 파일로 변환한다.
서버는 변환된 json파일을 메모리에 올려서 사용한다.

추가 개발 사항
서버가 사용할 수 있는 binary 파일로 변환 하면 더 좋다. (json과 크기 비교)
git 과 연동하여 커밋하고 로그 남기기

유의해야 할 사항
원본에 해당하는 기획 테이블의 형태는 일관적인 편이 좋다.
