개발 사항
제작된 패킷파일, json 데이터, 또는 공용코드를 특정 git의 주소로 업로드 하는 기능을 만든다.
기획자가 데이터 변환을 하거나 벡엔드 개발자가 패킷파일을 컨버팅 한 후 공용 작업 공간 git에 
push 하는 기능을 만든다.
일단은 초기 버전으로는 무조건 특정 git에 pull을 받고 push 하는 기능만 구현한다.
git에 push 후 로그를 남긴다.


추가 개발 사항
